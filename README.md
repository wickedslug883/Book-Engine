# Google Book Search 

## Summary

Google Api using GraphQl to grab the information from any book the user would like to search, as well as any other relevant information.

## Tools Used

* JavaScript
* Node.js
* Express
* React
* Apollo Server
* Graphql
* Mongodb
* Mongoose
* Dotenv
* Heroku
